# Project Brief

## Overview
BrowserRPG is a retro nation-building browser game with real-time multiplayer capabilities.

## Core Goals
- Build a 500x500 tile-based world where players can explore and conquer territory
- Enable players to create nations and build structures
- Implement real-time multiplayer with WebSocket connections
- Use retro 16-bit pixel art aesthetic

## Key Features
- Massive procedurally-generated world (250,000 tiles)
- 6 terrain types: Plain, Forest, Mountain, Desert, Water, Coast
- Nation building and territory conquest
- Real-time multiplayer competition
- Chunk-based world loading for performance

## Target Platform
- Web browser (no downloads required)
- Cross-platform compatible
- Responsive design for various screen sizes
